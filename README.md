# 🎵 RadioJS

**RadioJS** — це веб-додаток для легкого онлайн-прослуховування українських радіостанцій.

---

## 🟡 Особливості

- **Більше 20 українських радіостанцій** — сучасна та регіональна музика й новини.
- **Сучасний інтерфейс** — красивий відео-фон, приємний дизайн на основі CSS Grid.
- **Адаптивність** — зручно працює як на смартфоні, так і на комп’ютері.
- **Відтворення у 1 клік** — просто натискай на станцію, щоб слухати.
- **Немає реклами чи зайвих елементів**.

---

## 🔥 Початок роботи

1. **Скачайте або склонуйте репозиторій.**
2. Відкрийте `radio.html` у будь-якому браузері.
3. Клікайте по улюбленій радіостанції — і слухайте!

---

## ☑️ Stack

- **HTML**  
- **CSS** (у стилювальному блоці)
- **JavaScript** (інлайн для управління аудіо)
